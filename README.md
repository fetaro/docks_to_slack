# Google Docsの箇条書きをSlackにコピーするツール

## 課題

Google Docsの箇条書きをMacのクリップボードにコピーしてSlackにペーストすると、以下のように箇条書きが崩れてしまう。

Google Docsの箇条書き

---
・箇条書きレベル１
　　・箇条書きレベル２−１
　　・箇条書きレベル２−２
----

これをSlackにコピーすると

----
・箇条書きレベル１
・箇条書きレベル２−１
・箇条書きレベル２−２
---

とフラットな箇条書きでコピーされてしまう

### 目的

本ツールの目的は、この課題を解決することである。
MacOSのクリップボードにdocsの箇条書きがある状態でこのプログラムを実行すると、クリップボードの内容を取得し、Slackに正しくコピーできる文字列に変換し、クリップボードに格納する
